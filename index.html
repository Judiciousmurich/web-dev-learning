flexbox

for flexbox to work, we need to turn the parent container to a flex element

When we do display: flex; a main axis and cross axis are created in the element.

By default, the elements are arranged along the main axis. This is called flex-direction: row;

To make it go along the column the cross axis to make a column, we use flex-direction:column;
